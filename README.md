# E-Commerce Sales Analytics (Capstone Project)

##  Project Overview
This project is an end-to-end data analytics capstone based on real-world e-commerce transaction data. It focuses on analyzing sales performance, customer behavior, and revenue trends using Python-based data analysis techniques.

The project includes data cleaning, preprocessing, exploratory data analysis (EDA), and visualization to extract meaningful business insights.

---

##  Objectives
- Clean and preprocess raw e-commerce transaction data  
- Analyze overall sales trends and revenue patterns  
- Identify top-selling products and high-revenue items  
- Study country-wise sales contribution  
- Analyze time-based sales patterns (monthly and daily trends)  
- Generate actionable business insights  

---

##  Tools & Technologies
Python | Pandas | NumPy | Matplotlib | Seaborn | Jupyter Notebook  

---

##  Data Processing Steps
- Removed missing values and duplicate records  
- Filtered cancelled transactions  
- Converted data types for accurate analysis  
- Created new feature: **Total Sales = Quantity × Unit Price**  
- Extracted time-based features (Year, Month, Day)  

---

## Exploratory Data Analysis (EDA)
- Sales trend analysis over time  
- Top-selling products by quantity and revenue  
- Country-wise sales performance analysis  
- Monthly and seasonal sales trends  
- Customer purchasing behavior patterns  

---

##  Key Insights
- A small number of products generate the majority of revenue (Pareto principle)  
- Sales show strong seasonal and monthly variations  
- Certain countries dominate total sales contribution  
- Revenue is influenced by both product price and demand patterns  
- Business performance can be improved by focusing on top-performing products
